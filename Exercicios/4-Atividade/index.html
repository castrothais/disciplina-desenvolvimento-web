Desenvolvimento Web

HTML
Abreviação do termo inglês " HyperText Markup Language", que significa "Linguagem de marcação de texto".
O HTML é responsável pela estrutura da página e informa ao navegador como exibir o conteúdo.
CSS
Cascading Style Sheet, ou Folha de Estilo em Cascatas, é responsável por estilizar as páginas webs. Através do CSS, é possível alterar a cor de um texto, fonte, dentre outros.
JavaScript
É uma linguagem de programação interpretada estruturada e de tipagem fraca. Através dela, a página possui interatividade.
